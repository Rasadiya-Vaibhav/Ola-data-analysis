# Ola-data-analysis
Ola Ride Data Analysis & Dashboard 🚖
This repository contains a comprehensive data analytics project focused on Ola ride-booking data. It combines SQL-based data transformation with an interactive Power BI dashboard to deliver deep operational, financial, and behavioral insights.

📊 Project Overview & Key Insights
The project is structured into five core analytical workflows across both SQL and Power BI:


Overall Performance: Tracks high-level KPIs like Total Bookings, Total Booking Value, daily ride volumes, and overall booking status breakdowns.  


Vehicle Type Analysis: Compares metrics across vehicle categories (Mini, Auto, Prime, etc.), evaluating successful booking values, average distances, and rating variations.  



Revenue Insights: Analyzes financial performance from successful rides, ranking top customers and tracking revenue trends across payment options like Cash, Card, and UPI.  


Cancellation Breakdown: Isolates total cancellation rates and uses pie charts to drill down into customer triggers (e.g., change of plans) and driver reasons (e.g., car issues).  


Ratings & Sentiment: Provides a granular look at customer and driver sentiment, including minimum/maximum rating thresholds across different vehicle types.  


🛠️ Tools & Technologies Used
Database & SQL Analysis
Modular SQL Views and queries were built to clean, aggregate, and structure the raw ride data.  


State Control: CREATE DATABASE, USE   


Data Abstraction: CREATE VIEW for modular, reusable reporting   


Aggregations: SUM(), AVG(), COUNT(), MAX(), MIN()   


Data Structuring: GROUP BY, ORDER BY, LIMIT   

Power BI Business Intelligence
An interactive, multi-page Power BI Desktop report was developed for visual storytelling.  



Visualizations: KPI Cards, Donut/Pie Charts, Line Charts, Bar Charts, and Matrix Tables   


Features: Interactive navigation pane, date slicers, and cross-filtering capabilities
